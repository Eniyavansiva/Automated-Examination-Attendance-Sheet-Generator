Automated Examination Attendance Sheet Generator IGCSE grades which has multiple components for each subject and varied exam durations. The system reads exam schedule and student-list PDFs, then creates print-ready attendance sheets by grade, section, and exam session in less than a minute. It cuts teacher documentation effort from ~8–9 hours to under a minute (>99% time savings), reduces manual errors, and is designed for yearly reuse by simply updating the new academic-year PDFs.
