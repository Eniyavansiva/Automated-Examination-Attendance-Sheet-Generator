Designed an automated exam attendance sheet documentation system to handle the complexities using CODEX. It is for IGCSE grades which has multiple components for each subject. By extracting data from exam schedule and student-list PDFs, the system generates over 120 pages of print-ready attendance sheets per grade for grades 9 through 12. This automation reduces a tedious 8 to 9-hour manual workflow into a process that takes less than a minute. It eliminates manual data entry errors and is built for long-term scalability, requiring only updated PDFs to function in new academic years.
